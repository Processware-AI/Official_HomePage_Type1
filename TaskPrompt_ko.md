"Processware AI"라는 브랜드의 MVP 랜딩 웹사이트를 구축하고 있습니다.

목표는 잠재 고객 확보에 초점을 맞춘 깔끔하고 모던하며 반응형 디자인의 단일 페이지 웹사이트를 구현하는 것입니다.

과도한 설계는 금물입니다.
불필요한 백엔드 복잡성을 추가하지 마십시오.
완전한 SaaS 제품을 개발하지 마십시오.
이 프로젝트는 하나의 인터랙티브 추천 기능을 갖춘 랜딩 페이지 MVP입니다.

===================================================
1. 프로젝트 목표
=====================================================

다음과 같은 기능을 수행하는 홈페이지를 구축합니다.

1. Processware AI 브랜드를 명확하게 전달합니다.
2. 3가지 핵심 서비스(AI 어드바이저, AI Advisor, AI Advisor)를 설명합니다.
3. 대화형 "AI 어드바이저" 추천 도구를 제공합니다.
4. 문의 양식을 통해 방문자를 잠재 고객으로 전환합니다.

단순한 브로슈어 사이트가 아닙니다.

전환율 향상에 초점을 맞춘 랜딩 페이지입니다.

주요 KPI:
- 리드 전환율

보조 KPI:
- AI 어드바이저 사용률
- CTA 클릭률

===================================================
2. 브랜드 메시지
====================================================

브랜드명:
Processware AI

핵심 메시지:
“우리는 AI를 실질적인 업무로 전환합니다.”

3대 핵심 요소:
- 제품: AI 기반
- 프로세스: AI 중심
- 인력: AI 역량 강화

어조:

- 친근함
- 전문적
- B2B 신뢰도 높음
- 간결하지만 과하지 않음 콜드

7. 섹션 요구 사항
====================================================

------------------------------
7.1 히어로
------------------------------

목적:
브랜드 방향과 가치를 즉시 전달합니다.

내용:
제목:
“AI로 미래의 업무 환경을 구축하세요”

부제목:
“제품, 프로세스, 그리고 사람 – AI 시대에 맞춰 발전”

간략한 설명:
AI는 단순한 기술이 아니라, 업무 방식입니다.
Processware AI는 AI 시대를 위해 솔루션, 프로세스, 그리고 사람을 연결합니다.

CTA 버튼:
- “AI로 시작하기”
- “솔루션 살펴보기”

동작:
- 하나의 CTA 버튼을 클릭하면 AI 어드바이저 페이지로 이동합니다.
- 다른 하나의 CTA 버튼을 클릭하면 서비스 페이지로 이동합니다.

레이아웃:
- 왼쪽: 텍스트
- 오른쪽: 보기 좋고 밝은 시각적 영역
- 최종 디자인이 제공되지 않은 경우, 세련된 추상적인 임시 이미지를 사용할 수 있습니다.

------------------------------
7.2 핵심 메시지
------------------------------

내용:
“AI를 실질적인 업무로 전환합니다.”

동작:
- 크고 중앙에 배치된 타이포그래피
- 강력한 시각적 강조
- 주변 요소 최소화

------------------------------
7.3 세 가지 핵심 요소
------------------------------

반응형 카드 3개를 표시합니다.

카드 1:
제목: 제품
슬로건: AI 기반
설명: 실제 업무를 위한 AI 기반 솔루션

카드 2:
제목: 프로세스
슬로건: AI 주도
설명: AI로 재설계된 운영 시스템

카드 3:
제목: 인력
슬로건: AI 역량 강화
설명: 팀이 AI를 효과적으로 활용할 수 있도록 지원

동작:
- 반응형 레이아웃
- 마우스 오버 시 표시 기능
- 깔끔한 카드 디자인

------------------------------
7.4 서비스
------------------------------

간결한 설명과 함께 서비스 블록을 표시합니다.

서비스:
- AI 솔루션
- 프로세스 컨설팅
- AI 교육
- 플랫폼 개발

각 서비스에는 다음이 포함되어야 합니다.

- 제목
- 간략한 설명

너무 장황하게 작성하지 마십시오.

------------------------------
7.5 AI 어드바이저
------------------------------

이것은 MVP에서 가장 중요한 인터랙티브 기능입니다.

목적:
사용자 답변을 바탕으로 3가지 핵심 요소 중 하나를 추천합니다.

입력값:
질문 1: 가장 필요한 것은 무엇입니까?

- 솔루션
- 프로세스
- 교육

질문 2: 가장 큰 어려움은 무엇입니까?

- 자동화
- 규정 준수
- 역량

질문 3: 귀사는 어떤 유형의 조직인가요?

- 스타트업
- 대기업
- 규제 대상
- 팀 역량 강화

로직:
간단한 점수 기반 추천 로직을 사용합니다.

추천 매핑:
- 솔루션 => 제품
- 자동화 => 제품
- 스타트업 => 제품

- 프로세스 => 프로세스
- 규정 준수 => 프로세스
- 규제 대상 => 프로세스
- 대기업의 프로세스 개선

- 교육 => 인력
- 역량 => 인력
- 팀 역량 강화 => 인력

출력:
다음 내용을 포함하는 결과 카드를 표시합니다.
- 추천 핵심 요소
- 슬로건
- 맞춤형 간략 설명

예시 출력:
추천: 프로세스
슬로건: AI 기반
설명: 귀사는 구조화된 프로세스 혁신과 규정 준수 중심의 운영 설계를 통해 가장 큰 이점을 얻을 수 있을 것으로 예상됩니다.

결과 카드 내 CTA 버튼:
- 문의하기
- 자세히 알아보기

구현 참고 사항:
- MVP 단계에서는 로직을 로컬/클라이언트 측에서 처리하세요.
- 깔끔한 상태 관리를 사용하세요.
- 결과는 명확하고 보기 좋게 표시되어야 합니다.
- 제출 시 업데이트되거나 모든 필드를 선택한 후 실시간으로 업데이트됩니다.

------------------------------
7.6 CTA 섹션
------------------------------

내용:
“AI로 구축을 시작하세요”

간략한 설명:
AI를 활용하여 조직에서 실질적인 업무를 수행하세요.

버튼:
- 문의하기
- 데모 요청

------------------------------
7.7 문의 양식
------------------------------

입력 필드:
- 이름
- 이메일
- 회사명
- 메시지

유효성 검사:
- 이름 (필수)
- 이메일 (필수, 유효해야 함)
- 메시지 (필수)

제출:
MVP 단계에서는 가장 간단하고 실용적인 방법을 사용하세요.

선호하는 순서:
1. 웹훅 친화적인 구현
2. 간편한 이메일/API 통합
3. 통합이 불가능한 경우 로컬에서 모의 ​​성공 처리 구현

필요 사항:
- 명확한 성공 상태 표시
- 오류 처리
- 필요한 경우 성공 후 폼 초기화

8. 디자인 방향
===================================================

스타일:
- 밝은 색상
- 친근한 느낌
- 모던한 느낌
- 미니멀한 디자인
- B2B에 대한 신뢰감

색상:
- 흰색 배경
- 어두운 텍스트
- 브랜드에서 영감을 받은 주황색/빨간색 계열의 주요 색상
- 지나치게 어둡거나 공격적인 테마 지양

타이포그래피:
- 굵은 제목
- 깔끔한 서체 산세리프체 본문
- 여백이 넓은 레이아웃
- 높은 가독성

UI:
- 넓은 간격
- 적절한 경우 둥근 카드/버튼
- 부드러운 그림자
- 깔끔한 섹션 구성
